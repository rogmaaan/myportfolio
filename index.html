E-commerce Responsive

Objectif

Développer une plateforme e-commerce complète avec gestion de produits, panier d'achat et système de paiement sécurisé.

Technologies

  • • PHP 8 / Laravel
  • • JavaScript / jQuery
  • • MySQL
  • • HTML5 / CSS3 / Bootstrap
  • • API Stripe (paiement)

Compétences

  • • Compétence A (Développer)
  • • Compétence C (Administrer)
  • • Compétence E (Conduire)

Description détaillée

Ce projet a été réalisé dans le cadre de la SAÉ 5.01. Il s'agit d'une plateforme e-commerce complète permettant la gestion de produits, catégories, utilisateurs et commandes.

Fonctionnalités principales :

  • Catalogue de produits avec filtres et recherche
  • Système de panier d'achat avec persistance
  • Processus de commande et paiement sécurisé
  • Espace administrateur pour la gestion des produits et commandes
  • Système de notation et commentaires pour les produits

Le projet a été développé en suivant une méthodologie Agile avec des sprints de deux semaines et des réunions hebdomadaires avec le client.

Application Mobile Fitness

Objectif

Créer une application mobile de suivi d'activités sportives avec des plans d'entraînement personnalisés et des statistiques détaillées.

Technologies

  • • Java / Kotlin
  • • Android SDK
  • • Firebase (Auth, Realtime DB)
  • • MPAndroidChart
  • • Google Fit API

Compétences

  • • Compétence A (Développer)
  • • Compétence B (Optimiser)
  • • Compétence D (Gérer)

Description détaillée

Ce projet personnel a été développé pour répondre à un besoin de suivi d'activités sportives plus complet que les applications existantes sur le marché.

Fonctionnalités principales :

  • Suivi des séances d'entraînement avec détails des exercices
  • Création de plans d'entraînement personnalisés
  • Statistiques et graphiques d'évolution
  • Intégration avec les capteurs du téléphone (podomètre, GPS)
  • Synchronisation des données entre appareils

L'application a été conçue en suivant les principes de Material Design et optimisée pour offrir une expérience utilisateur fluide et intuitive.

Dashboard Analytics

Objectif

Développer un tableau de bord interactif pour visualiser et analyser des données commerciales en temps réel.

Technologies

  • • React.js
  • • Node.js / Express
  • • MongoDB
  • • D3.js / Chart.js
  • • Socket.io (données temps réel)

Compétences

  • • Compétence A (Développer)
  • • Compétence B (Optimiser)
  • • Compétence F (Collaborer)

Description détaillée

Ce projet a été réalisé dans le cadre de la SAÉ 4.01 en collaboration avec une entreprise locale qui souhaitait améliorer le suivi de ses performances commerciales.

  • Alertes et notifications en temps réel

Le dashboard a été conçu avec une architecture modulaire permettant d'ajouter facilement de nouvelles visualisations et fonctionnalités selon les besoins évolutifs de l'entreprise.

Gestionnaire de Tâches Collaboratif

Objectif

Développer une application web permettant la gestion de projets en équipe avec attribution de tâches et suivi d'avancement.

Technologies

  • • Vue.js
  • • Express.js
  • • PostgreSQL
  • • Socket.io
  • • JWT (authentification)

Compétences

  • • Compétence A (Développer)
  • • Compétence E (Conduire)
  • • Compétence F (Collaborer)

Description détaillée

Ce projet a été réalisé dans le cadre de la SAÉ 3.01 en équipe de 4 étudiants. L'objectif était de créer une alternative à Trello avec des fonctionnalités supplémentaires adaptées aux besoins des équipes de développement.

Fonctionnalités principales :

  • Tableau Kanban avec colonnes personnalisables
  • Gestion des utilisateurs et des équipes
  • Système de commentaires et de pièces jointes
  • Notifications en temps réel
  • Suivi du temps passé sur chaque tâche
  • Rapports et statistiques d'avancement

L'application a été développée en suivant une architecture MVC et des principes de conception SOLID pour assurer une maintenance facile et une évolutivité du code.
